## 🔧 Customization

### Want to Change the Gold Color?
Edit `src/App.css`:
```css
:root {
  --accent: #YOUR-COLOR;
  --accent-hover: #DARKER-VERSION;
  --accent-light: #LIGHTER-VERSION;
}
```

### Want Different Warm Base Colors?
Edit `src/App.css`:
```css
:root {
  --bg: #YOUR-BG;
  --surface: #YOUR-SURFACE;
  --border: #YOUR-BORDER;
}
```
