# Aurevia Estates & Development Ltd.

Marketing website for **Aurevia Estates & Development Ltd.** — a premium real estate and property development company based in Victoria Island, Lagos.

**Tagline:** Building Tomorrow's Landmarks Today.

## About

Aurevia specializes in luxury residential developments, commercial real estate, property investment advisory, facility management, and land acquisition across emerging urban markets in Africa.

## Features

- Premium landing page with property search
- Core services, portfolio, and location showcases
- Mission, vision, and company overview
- Investment advisory packages
- Contact form and newsletter signup
- Fully responsive layout

## Tech Stack

- React 18
- React Router
- CSS (custom design system)
- Create React App

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```

Production files are output to the `build/` directory.

## Deploy

This project includes a `vercel.json` for deployment on [Vercel](https://vercel.com). Connect the repository and deploy, or run:

```bash
npx vercel
```
